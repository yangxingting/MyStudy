Vue-Router

Vue-Router是用来将一个Vue程序的多个页面进行路由的。比如一个Vue程序（或者说一个网站）有登录、注册、首页等模块，那么我们就可以定义/login、/register、/来映射每个模块。

安装：

  1. 通过script加载进来：<script src="https://unpkg.com/vue-router/dist/vue-router.js"></script>。
  2. 通过npm安装：npm install vue-router。

文档：

  1. 官方文档：https://router.vuejs.org/zh/。
  2. github地址：https://github.com/vuejs/vue-router。

版本：

本课程用到的最新的版本是：3.1.3。
