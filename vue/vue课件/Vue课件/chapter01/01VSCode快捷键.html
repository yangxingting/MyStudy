VSCode快捷键表：

快捷键有五种组合方式(科普)

  1. Ctrl + Shift + ?：这种常规组合按钮
  2. Ctrl + C Ctrl +V：同时依赖一个按键的组合
  3. Shift + V C：先组合后单键的输入
  4. Ctrl + Click： 键盘 + 鼠标点击
  5. Ctrl + DragMouse： 键盘 + 鼠标拖动 macos下大多键位等同，Ctrl换成Command

通用快捷键

快捷键 作用
Ctrl+Shift+P,F1 展示全局命令面板
Ctrl+P 快速打开最近打开的文件
Ctrl+Shift+N 打开新的编辑器窗口
Ctrl+Shift+W 关闭编辑器
